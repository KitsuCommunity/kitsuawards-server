# Kitsu Awards

This is the server backend to the Kitsu Awards.

## Requirements

- Ruby 3.0.x
- Node.js 18.x
- PostgreSQL 12.x
- Redis 7.x (_installed during setup in development_)
- Memcached 1.x (_installed during setup in development_)
